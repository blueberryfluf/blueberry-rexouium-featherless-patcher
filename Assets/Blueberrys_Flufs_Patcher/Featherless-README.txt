Menu
----
Tools → BlueBerry → Rexouium Featherless Fox Patch

What you get (keeps fluff!)
---------------------------
FUN
• Normal — resets ALL blueberry fox options back to defaults
• Moods: Happy / Smug / Scowl / Blep
• Stare: Prey Panic / Predator
• Puff & Fluff emote (boosts fluff — does NOT remove feathers/fluff)
• Tail: Pose → Rest / Lower (lay) / Up / Tucked + Thick / Plume / Thin + Wag Speed
  - Rest = physbones on, fluff settles like stock
  - Tucked = stock tucked pose with physbones on
  - Lower pins the tail aft toward the feet and turns off tail physbones
    so it won't flop into your face while prone
• Derp: Wa / Niyari / Pero / Wao?!

QoL
• Fur density: Stock / Soft / Fluffy / Sleek
• Body radials: Weight, Muscle, Breasts, Feminine, Paw Size
• Face: Muzzle Short / Long / Thin
• Idle: Breathing on/off + Breath Speed, Nose Twitch on/off
  (Body + Idle/Face live under More)

Requirements
------------
1. Rexouium Evolved RoyaltyFT (Featherless prefab)
2. VRCFury (already required by RoyaltyFT face tracking)
3. Body mesh path named "Body" with stock Rex blendshapes

Usage
-----
1. Tools → BlueBerry → Rexouium Featherless Fox Patch
2. Click "1. Build / Refresh Patch Assets"
3. Drag your Featherless avatar root into Target Avatar
4. Click "2. Patch Avatar"
5. Find menu: blueberry fox / …

Notes
-----
• Patch adds a child object "BlueBerry Fox Patch" with a VRCFury Full Controller.
• Re-running Patch replaces that child (safe to apply again).
• Breathing speed radial: slow ←→ fast (about 0.35x to 2x).
• Parameter memory uses several BB* synced params — check VRChat param budget if you already have a huge menu.
