## What this repo includes

- `Assets/Blueberrys_Flufs_Patcher/Editor/RexFeatherlessPatcher.cs` — Editor window
- `Assets/Blueberrys_Flufs_Patcher/Editor/Icons/` — Discord / GitHub / Instagram / community icons for the window
- README / Featherless-README

**Not included:** Rexouium / RoyaltyFT game assets, generated FX clips, menus, or controllers (those are built inside Unity).

## Menu

**Tools → BlueBerry → Rexouium Featherless Fox Patch**

### What you get (keeps fluff!)

**FUN**
- Normal — resets ALL blueberry fox options back to defaults
- Moods: Happy / Smug / Scowl / Blep
- Stare: Prey Panic / Predator
- Puff & Fluff emote (boosts fluff — does NOT remove feathers/fluff)
- Tail: Pose → Rest / Lower (lay) / Up / Tucked + Thick / Plume / Thin + Wag Speed
- Derp: Wa / Niyari / Pero / Wao?!

**QoL**
- Fur density: Stock / Soft / Fluffy / Sleek
- Body radials: Weight, Muscle, Breasts, Feminine, Paw Size
- Face: Muzzle Short / Long / Thin
- Idle: Breathing on/off + Breath Speed, Nose Twitch on/off

## Requirements

1. Rexouium Evolved RoyaltyFT (Featherless prefab)
2. VRCFury (already required by RoyaltyFT face tracking)
3. Body mesh path named `Body` with stock Rex blendshapes

## Usage

1. Import / copy this folder into your Unity VRChat avatar project
2. **Tools → BlueBerry → Rexouium Featherless Fox Patch**
3. Click **1. Build / Refresh Patch Assets**
4. Drag your Featherless avatar root into Target Avatar
5. Click **2. Patch Avatar**
6. Find menu: `blueberry fox / …`
